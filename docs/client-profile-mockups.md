### Photo Upload Flow:
1. User taps "Cambiar foto" or camera icon
2. Modal opens with upload area
3. User selects/drops photo
4. Image validates (type, size)
5. Crop interface opens automatically
6. User adjusts crop and zoom
7. Tap "Usar esta foto"
8. Image uploads to S3 (show progress)
9. Profile updates with new URL
10. Modal closes, photo appears in profile
